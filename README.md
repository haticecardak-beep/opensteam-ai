# opensteam-ai
OpenSTEAM AI is an open-source AI toolkit that enables teachers to generate lesson plans, STEAM activities, and cultural heritage content.  It provides a modular and reusable infrastructure using open AI models, empowering teachers as content creators and supporting inclusive, transparent, and open digital education.
# OpenSTEAM AI

OpenSTEAM AI is an open-source AI-powered toolkit designed to support teachers in creating STEM and cultural heritage-based learning content.

## Vision
To empower teachers as content creators using open and transparent AI tools.

## Core Features
- AI Lesson Plan Generator
- STEAM Activity Builder
- Cultural Heritage Content Module
- Sustainability Awareness Content Generator

## Technical Approach
- Modular architecture
- Open-source AI models integration
- Export formats (PDF, JSON, AR/3D-ready)

## Roadmap
- Phase 1: Core system design
- Phase 2: AI integration
- Phase 3: Testing with teachers
- Phase 4: Open-source release
